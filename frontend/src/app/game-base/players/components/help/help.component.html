Controls

Plane steering

W/↑ - Accelerate

S/↓ - Decelerate

A/← - Turn left

D/→ - Turn right

Camera steering

C - Switch camera mode

V - Switch followed player

Airport controls

F - Land on the airport

A/← - Move to package on the left

D/→ - Move to package on the right

↩ - Select package

F - Take off

X - Refuel plane
